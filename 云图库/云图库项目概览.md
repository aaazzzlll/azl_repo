# 应用启动的完整调用链
```text
index.html
  └─ <script src="/src/main.ts">
       ├─ createApp(App.vue)
       ├─ app.use(Pinia)          ← 全局状态管理
       ├─ app.use(Router)         ← 路由注册
       ├─ app.use(Antd)           ← UI库注册
       ├─ app.use(VueCropper)     ← 裁切库
       ├─ import '@/access'       ← 全局权限守卫（副作用导入）
       └─ app.mount('#app')
            └─ App.vue
                 └─ <BasicLayout />
                      ├─ <GlobalHeader />   ← 顶部导航栏
                      ├─ <GlobalSider />    ← 左侧菜单栏
                      └─ <router-view />    ← 页面内容区（由路由决定渲染哪个页面）
```
