# 应用启动的完整调用链
```text
index.html
  └─ <script src="/src/main.ts">
       ├─ createApp(App.vue)
       ├─ app.use(Pinia)          ← 全局状态管理
       ├─ app.use(Router)         ← 路由注册
       ├─ app.use(Antd)           ← UI库注册
       ├─ app.use(VueCropper)     ← 裁切库
       ├─ import '@/access'       ← 全局权限守卫（副作用导入）
       └─ app.mount('#app')
            └─ App.vue
                 └─ <BasicLayout />
                      ├─ <GlobalHeader />   ← 顶部导航栏
                      ├─ <GlobalSider />    ← 左侧菜单栏
                      └─ <router-view />    ← 页面内容区（由路由决定渲染哪个页面）
```

# 核心架构分层
```text
┌─────────────────────────────────────────────────────┐
│                    页面层 (pages/)                   │
│  HomePage / PictureDetailPage / SpaceDetailPage ... │
├─────────────────────────────────────────────────────┤
│                   组件层 (components/)               │
│  PictureList / PictureSearchForm / ShareModal ...   │
├─────────────────────────────────────────────────────┤
│                   状态层 (stores/)                   │
│  useLoginUserStore (Pinia)                          │
├─────────────────────────────────────────────────────┤
│                   API 层 (api/)                     │
│  userController / pictureController / space... 等   │
├─────────────────────────────────────────────────────┤
│                   请求层 (request.ts)                │
│  Axios 实例 + 请求/响应拦截器                          │
├─────────────────────────────────────────────────────┤
│                  基础设施层                           │
│  router/ (路由)  access.ts (权限)  utils/ (工具)      │
└─────────────────────────────────────────────────────┘
```
