# 应用启动的完整调用链
```text
index.html
  └─ <script src="/src/main.ts">
       ├─ createApp(App.vue)
       ├─ app.use(Pinia)          ← 全局状态管理
       ├─ app.use(Router)         ← 路由注册
       ├─ app.use(Antd)           ← UI库注册
       ├─ app.use(VueCropper)     ← 裁切库
       ├─ import '@/access'       ← 全局权限守卫（副作用导入）
       └─ app.mount('#app')
            └─ App.vue
                 └─ <BasicLayout />
                      ├─ <GlobalHeader />   ← 顶部导航栏
                      ├─ <GlobalSider />    ← 左侧菜单栏
                      └─ <router-view />    ← 页面内容区（由路由决定渲染哪个页面）
```

# 核心架构分层
```text
┌─────────────────────────────────────────────────────┐
│                    页面层 (pages/)                   │
│  HomePage / PictureDetailPage / SpaceDetailPage ... │
├─────────────────────────────────────────────────────┤
│                   组件层 (components/)               │
│  PictureList / PictureSearchForm / ShareModal ...   │
├─────────────────────────────────────────────────────┤
│                   状态层 (stores/)                   │
│  useLoginUserStore (Pinia)                          │
├─────────────────────────────────────────────────────┤
│                   API 层 (api/)                     │
│  userController / pictureController / space... 等   │
├─────────────────────────────────────────────────────┤
│                   请求层 (request.ts)                │
│  Axios 实例 + 请求/响应拦截器                          │
├─────────────────────────────────────────────────────┤
│                  基础设施层                           │
│  router/ (路由)  access.ts (权限)  utils/ (工具)      │
└─────────────────────────────────────────────────────┘
```

# 六大业务模块
## 用户模块
```text
UserLoginPage ──→ userLoginUsingPost() ──→ /api/user/login
      │                                        ↓
      └── 登录成功 → loginUserStore.setLoginUser() → 跳转首页

UserRegisterPage ──→ userRegisterUsingPost() ──→ /api/user/register
      │
      └── 注册成功 → 跳转登录页

GlobalHeader ──→ userLogoutUsingPost() ──→ /api/user/logout
      │
      └── 退出成功 → 清空 store → 跳转登录页
```
## 图片模块
```text
HomePage (公共图库浏览)
  └─ PictureSearchForm (搜索表单) ──→ listPictureVoByPageUsingPost()
  └─ PictureList (图片列表) ──→ 点击 → PictureDetailPage

AddPicturePage (上传/编辑图片)
  ├─ PictureUpload (文件上传) ──→ uploadPictureUsingPost()
  ├─ UrlPictureUpload (URL上传) ──→ uploadPictureByUrlUsingPost()
  ├─ ImageCropper (裁切) ──→ uploadPictureUsingPost() + WebSocket协同
  └─ ImageOutPainting (AI扩图) ──→ createPictureOutPaintingTask → 轮询 → uploadByUrl

AddPictureBatchPage (批量抓取)
  └─ uploadPictureByBatchUsingPost() ──→ 后端爬取图片

PictureDetailPage (详情页)
  └─ 查看/下载/分享/编辑/删除
```
## 空间模块
```text
AddSpacePage (创建空间)
  └─ addSpaceUsingPost() ──→ 创建后跳转 /my_space

MySpacePage (入口中转页)
  └─ 查询用户私有空间 → 有则跳转 SpaceDetailPage，无则跳转 AddSpacePage

SpaceDetailPage (空间详情)
  ├─ PictureSearchForm (搜索)
  ├─ PictureList (图片列表)
  └─ BatchEditPictureModal (批量编辑) ──→ editPictureByBatchUsingPost()
```
## 团队协作模块
```text
GlobalSider 侧边栏
  └─ listMyTeamSpaceUsingPost() → 展示用户加入的团队空间

SpaceUserManagePage (成员管理)
  ├─ 添加成员 → addSpaceUserUsingPost()
  ├─ 编辑角色 → editSpaceUserUsingPost()  (viewer/editor/admin)
  └─ 删除成员 → deleteSpaceUserUsingPost()

ImageCropper (实时协同)
  └─ PictureEditWebSocket → ws://localhost:8123/api/ws/picture/edit
      ├─ ENTER_EDIT / EXIT_EDIT (进入/退出编辑)
      └─ EDIT_ACTION (缩放/旋转操作实时同步)
```
## 数据分析模块
```text
SpaceAnalyzePage
  ├─ SpaceUsageAnalyze    → 仪表盘：存储/数量使用率
  ├─ SpaceCategoryAnalyze → 柱状图：分类图片分布
  ├─ SpaceTagAnalyze      → 词云图：标签使用频率
  ├─ SpaceSizeAnalyze     → 饼图：图片大小分布
  ├─ SpaceUserAnalyze     → 折线图：用户上传趋势
  └─ SpaceRankAnalyze     → 柱状图：空间使用量排行 Top10
```
## 管理员后台模块
```
/admin/userManage    → 用户管理：搜索、删除用户
/admin/pictureManage → 图片管理：搜索、审核（通过/拒绝）、编辑、删除
/admin/spaceManage   → 空间管理：搜索、删除空间
```
