# 应用启动的完整调用链
```text
index.html
  └─ <script src="/src/main.ts">
       ├─ createApp(App.vue)
       ├─ app.use(Pinia)          ← 全局状态管理
       ├─ app.use(Router)         ← 路由注册
       ├─ app.use(Antd)           ← UI库注册
       ├─ app.use(VueCropper)     ← 裁切库
       ├─ import '@/access'       ← 全局权限守卫（副作用导入）
       └─ app.mount('#app')
            └─ App.vue
                 └─ <BasicLayout />
                      ├─ <GlobalHeader />   ← 顶部导航栏
                      ├─ <GlobalSider />    ← 左侧菜单栏
                      └─ <router-view />    ← 页面内容区（由路由决定渲染哪个页面）
```


# 六大业务模块
## 用户模块
```text
UserLoginPage ──→ userLoginUsingPost() ──→ /api/user/login
      │                                        ↓
      └── 登录成功 → loginUserStore.setLoginUser() → 跳转首页

UserRegisterPage ──→ userRegisterUsingPost() ──→ /api/user/register
      │
      └── 注册成功 → 跳转登录页

GlobalHeader ──→ userLogoutUsingPost() ──→ /api/user/logout
      │
      └── 退出成功 → 清空 store → 跳转登录页
```
## 图片模块
```text
HomePage (公共图库浏览)
  └─ PictureSearchForm (搜索表单) ──→ listPictureVoByPageUsingPost()
  └─ PictureList (图片列表) ──→ 点击 → PictureDetailPage

AddPicturePage (上传/编辑图片)
  ├─ PictureUpload (文件上传) ──→ uploadPictureUsingPost()
  ├─ UrlPictureUpload (URL上传) ──→ uploadPictureByUrlUsingPost()
  ├─ ImageCropper (裁切) ──→ uploadPictureUsingPost() + WebSocket协同
  └─ ImageOutPainting (AI扩图) ──→ createPictureOutPaintingTask → 轮询 → uploadByUrl

AddPictureBatchPage (批量抓取)
  └─ uploadPictureByBatchUsingPost() ──→ 后端爬取图片

PictureDetailPage (详情页)
  └─ 查看/下载/分享/编辑/删除
```
