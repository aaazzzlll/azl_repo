### 一、 📍 导航类：用来“走路”的

在命令行里，你就是个探险家，需要知道你在哪，以及怎么去别的地方。

|**指令**|**英文全称**|**作用**|**举例**|
|---|---|---|---|
|**`ls`**|List|**看看这里有什么**。列出当前文件夹下的所有文件。|`ls`|
|**`cd`**|Change Directory|**进入某个文件夹**。|`cd src` (进入 src 文件夹)|
|**`cd ..`**|-|**后退一步**。返回上一级文件夹。|`cd ..`|
|**`pwd`**|Print Working Directory|**我在哪？** 显示当前完整路径。|`pwd`|

> 💡 必学技巧：Tab 键自动补全
> 
> 千万不要手动打完整的文件名！
> 
> 比如你想进入 D:\Documents，你只需要输入 cd D，然后猛按 Tab 键，系统会自动帮你补全剩下的名字。这是命令行最快的功能！

### 二、 🛠️ 操作类：用来“动手”的

这部分用来增删改查文件，就是你刚才问的“怎么创建文件”。

|**指令**|**作用**|**举例**|**备注**|
|---|---|---|---|
|**`mkdir`**|**创建文件夹** (Make Directory)|`mkdir my-project`|创建一个叫 my-project 的文件夹|
|**`ni`**|**创建文件** (New Item)|`ni index.js`|PowerShell 专用，创建一个空文件|
|**`rm`**|**删除文件/文件夹** (Remove)|`rm test.js`|**⚠️ 警告：** 命令行删除不进回收站，直接消失！慎用！|
|**`mv`**|**移动/重命名** (Move)|`mv old.js new.js`|把 old.js 改名为 new.js|
|**`cp`**|**复制** (Copy)|`cp a.js b.js`|把 a 复制一份变成 b|

### 三、 🚀 开发神器：前端必须会的

这两个命令能极大提高你的开发效率，一定要背下来。

#### 1. `code .` (召唤 VS Code)

这是一个魔法命令。

- **用法：** 在终端里进入一个项目文件夹，输入 `code .` 并回车。
    
- **效果：** VS Code 会自动打开，并且**直接以当前文件夹作为工作区**。再也不用打开软件然后点“文件 -> 打开文件夹”了。
    

#### 2. `explorer .` (召唤文件资源管理器)

- **用法：** 输入 `explorer .`
    
- **效果：** 会弹出一个 Windows 文件夹窗口，显示当前目录的内容。当你需要在图形界面里查看文件时很有用。

### 四、 🛑 控制类：关键时刻救命的

| **按键/指令**      | **作用**                | **场景**                                    |
| -------------- | --------------------- | ----------------------------------------- |
| **`Ctrl + C`** | **强制停止**              | 如果你的代码写了死循环，或者 Node 服务器想关掉，按这个！(有时候需要按两下) |
| **`cls`**      | **清屏** (Clear Screen) | 屏幕字太多看着心烦？输入 `cls` 回车，瞬间清空。               |
| **`↑` / `↓`**  | **历史记录**              | 按“上箭头”，可以自动填入你**上一条**执行过的命令。不想重复打字时非常有用。  |
