# 2026年前端学习路线

## 一、核心基础 (Core Foundations)

1.  **HTML & CSS:**
    *   深入理解语义化HTML标签和SEO基础。
    *   掌握现代CSS特性，包括Flexbox、Grid布局、响应式设计（rem, vw, 容器查询）。
    *   关注CSS新特性，如容器查询 (Container Queries)、嵌套规则 (Nesting)、CSS锚点定位 (Anchor Positioning) 等。
    *   了解CSS预处理器（如Sass/Less），但原生CSS的进步使其重要性逐渐降低。
    *   熟悉CSS框架，如Tailwind CSS或UnoCSS。

2.  **JavaScript (ES6+) & TypeScript:**
    *   精通JavaScript核心概念，包括作用域、执行上下文、原型链、闭包、异步编程（Promise, async/await, 事件循环）。
    *   掌握ES6+新特性。
    *   **TypeScript** 已成为前端开发的普遍标准，是大型项目和团队协作的必备技能。

## 二、主流框架与库 (Key Frameworks & Libraries)

1.  **React (首选)**:
    *   深入学习React Hooks全家桶及自定义Hook封装。
    *   关注React 18/19的并发特性 (Concurrent Features)。
    *   理解 **React Server Components (RSC)**。
    *   了解 **React Compiler** (React Forget)。

2.  **Next.js (全栈利器)**:
    *   掌握 Next.js 的 App Router、Server Components、RSC、服务端渲染 (SSR)、静态生成 (SSG) 和 API 路由等特性。
    *   Next.js 推动了“全栈前端”的转型。

3.  **Vue.js (重要替代)**:
    *   Vue 3 仍是重要的前端框架，建议作为第二技能学习。

## 三、状态管理 (State Management)

*   轻量级的状态管理库如 Zustand 或利用 React Compiler 优化后的 Context API 成为趋势。

## 四、构建工具与工程化 (Build Tools & Engineering)

1.  **构建工具:**
    *   **Vite** 已成为现代前端项目的默认选择。
2.  **包管理器:** npm, pnpm, yarn。
3.  **版本控制:** Git 是必备技能。
4.  **浏览器开发者工具:** 熟练使用进行调试和性能优化。
5.  **测试工具:** Jest, Vitest, Cypress, Testing Library 等。
6.  **性能优化:** 掌握 Core Web Vitals、代码分割、缓存、图片优化等技术。
7.  **CI/CD:** 自动化测试和部署流程，如使用 GitHub Actions。

## 五、全栈与部署 (Full-stack & Deployment)

*   理解全栈开发理念，尤其是 Next.js 提供的全栈能力。
*   熟悉主流部署平台，如 Vercel, Netlify, Cloudflare Pages, Firebase Hosting。
*   掌握 SEO 基础知识。
*   了解 PWA (Progressive Web Apps)。

## 六、新兴与高级主题 (Emerging & Advanced Topics)

*   **AI 辅助开发**
*   **原生组件开发**
*   **GraphQL**
*   **WebSockets**
*   **设计系统 (Design Systems)**
*   **微前端 (Microfrontends)**
*   **Web3/区块链**
*   **3D 渲染 (WebGL / Three.js)**

## 七、学习方法与心态 (Learning Approach)

*   **实践驱动:** 通过手写项目来巩固所学知识。
*   **官方文档:** 优先学习官方文档。
*   **持续学习:** 保持学习热情和适应能力。
*   **计算机基础:** 具备计算机网络、数据结构和算法、命令行操作等基本常识。
