1. 浏览器首先会**解析 HTML，生成 DOM 树**；同时**解析 CSS，生成 CSSOM 树**。
2. 然后把 DOM 树和 CSSOM 树**合并成渲染树 Render Tree**，渲染树只保留页面上可见的元素。
3. 接下来进行**布局 Layout，也叫回流**：计算每个元素在页面上的位置和大小。
4. 然后进行**绘制 Paint，也叫重绘**：把元素的颜色、背景、文字等画出来。
5. 最后一步是**合成 Composite**：把多个图层合并，最终显示到屏幕上。

另外：
- **JS 会阻塞 HTML 解析**，CSS 会阻塞渲染；
- **回流成本 > 重绘成本 > 合成**；
- 用 transform、opacity 只会触发合成，性能最好。
