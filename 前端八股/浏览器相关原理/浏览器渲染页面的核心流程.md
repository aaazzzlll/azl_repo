# 从HTML到DOM
浏览器通过网络进程拿到数据后，找到对应的渲染进程，把数据传输进去，渲染进程拿到的是字节数据
解析过程中若遇到`<script>`标签（无 async/defer），会暂停 DOM 解析，先执行 JS（因为 JS 可能修改 DOM），导致渲染阻塞
