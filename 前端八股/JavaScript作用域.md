1. JavaScript的作用域分为以下三种：
- 全局作用域：在浏览器环境下，全局变量通常挂载在 `window` 对象上
- 函数作用域
- 块级作用域

| 关键字   | 作用域类型       | 是否存在变量提升     | 是否允许重复声明 |
| ----- | ----------- | ------------ | -------- |
| var   | 函数作用域/全局作用域 | 是（提升到顶部但未赋值） | 是        |
| let   | 块级作用域       | 否（存在暂时性死区）   | 否        |
| const | 块级作用域       | 否（存在暂时性死区）   | 否        |
